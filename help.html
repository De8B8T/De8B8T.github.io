Current Commands

-info

Displays information about the bot, including the number of successful deliveries.

-cmds

Provides a small list of available commands.

-uptime

Returns the bot's uptime.

-kit [name]

Delivers a fast kit based on the specified name. Aliases available.

e.g. -kit pvp

-kits

Returns a list of all kits

-kithelp [pvp/build]

The -kithelp command provides a list of available kits and their aliases. Here are the kits you can request:

**PVP Kits:** - pvp (aliases: pvp, spvp) - hpvp (aliases: health, hpvp, totems, totem, gaps) - cpvp (aliases: crystalpvp, cpvp) - xp (aliases: xp, mend, experience) - refill (aliases: refill, armorrefill) - amrefill (aliases: amrefill, ammorefill) - elytra (aliases: elytra, ely, flight)

**Build Kits:** - wood (aliases: wood, cwood, signs) - stone (aliases: build, stone, cstone) - nether (aliases: nethermix, cnether, nether) - concrete (aliases: cconcrete, concrete, misc) - dupe (aliases: duplication, zombie, dupe) - redstone (aliases: redstone, red, electric)

-history

View the last 10 kits you requested.

-is [username]

Checks if a player is online or offline based on their username.

e.g. -is PlayerOne

-remindme [num (Minutes)] [Text]

Waits [NUM] before Reminding you [TEXT].

-feedback / -suggest

Suggest or give Feedback on the bot.

-roll [num1] [num2]

Private messages you a random number from [num1] to [num2] (like rolling a dice).

e.g. -roll 1 6

-fact

Private messages you a random fact from the API.

-quote

Private messages you a random quote from the API.

-flip

Private messages you a random side of a coin (heads/tails).

-joke

Returns a joke.

-tip

Returns a tip.
